# APLIKASI ANTRIAN

<p>Aplikasi ini dibangun menggunakan electron js, dan menggukana socket.io untuk mengirimkan antrian ke tampilan utama.</p>

## Run :
1. NPM Install
```javascript
npm i
```
2. Run Development
```javascript
npm run start:dev
```
